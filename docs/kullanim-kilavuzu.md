# Ağ Tarayıcı Kullanım Kılavuzu

## Parametreler
Sistem `src/network_scanner.py` dosyası üzerinden çalışır. Çalışma zamanı ayarlarını kök dizindeki `.env` dosyasından okur.

- `TARGET_IP`: Hedef sistemi belirler.
- `SCAN_TIMEOUT`: Yanıt beklerken geçecek süreyi saniye cinsinden ayarlar.

Daha fazla geliştirme detayı için ana dizindeki README.md dosyasına bakınız.
